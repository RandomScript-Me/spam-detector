How it works

Algorithm
Logistic Regression — a classic, interpretable ML model that learns a weighted combination of word features to classify text as spam or ham.
Feature extraction
TF-IDF Vectorization with bigrams (1–2 word phrases). Captures phrases like "click here" and "free prize" that single words miss.
Dataset
UCI SMS Spam Collection — 5,572 real SMS messages (4,825 ham, 747 spam). A benchmark dataset widely used in NLP research.
Performance
96% overall accuracy · 99% spam precision · Ham recall near perfect. Decision threshold tuned to 30% for aggressive spam catching.
